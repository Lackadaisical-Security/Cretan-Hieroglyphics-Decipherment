### Europe's Earliest Writing System Decoded

Cretan Hieroglyphic (c. 2100–1700 BCE) is now understood as a **logo-syllabic administrative script** used in Minoan palatial administration. It represents the parent system of Linear A and the ancestor of all later Aegean scripts.
